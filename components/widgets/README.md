# widgets

Widget implementations for ToonTalk objects.

## Crates

- **tt-rs-number**: Rational number widget with arithmetic operators
- **tt-rs-text**: Text/string widget
- **tt-rs-box**: Box container widget
- **tt-rs-scales**: Balance scales widget
- **tt-rs-vacuum**: Vacuum eraser tool
- **tt-rs-wand**: Copy wand tool
- **tt-rs-robot**: Programmable robot agent

## Dependency Level

Depends on: core
