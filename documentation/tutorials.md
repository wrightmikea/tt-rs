# Tutorials

Step-by-step guided lessons to learn tt-rs concepts.

## Overview

Tutorials are designed to introduce users to tt-rs features progressively. Each tutorial focuses on a single concept and builds on previous lessons.

## Proposed Tutorials

### Basic Level (tt1)

#### 1. Hello Numbers
**Goal**: Learn to create and manipulate numbers
**Concepts**: Number stacks, copying, basic arithmetic
**Steps**:
1. Drag a number from the palette to the workspace
2. Click and drag from a number to create a copy
3. Drop a number onto another to perform arithmetic
4. Try all four operations: +, -, *, /

**Expected Outcome**: User understands number stacks and arithmetic operations

#### 2. Boxing Things Up
**Goal**: Learn to use boxes as containers
**Concepts**: Box creation, holes, putting items in boxes
**Steps**:
1. Drag a box from the palette
2. Notice the holes in the box
3. Drag a number into a hole
4. Create a box with multiple holes (drop number on box edge)
5. Try removing items with the vacuum

**Expected Outcome**: User can store and organize items in boxes

#### 3. Balancing Act
**Goal**: Learn to compare values with scales
**Concepts**: Scales, comparison, tipping behavior
**Steps**:
1. Drag a scale from the palette
2. Drop a number on the left pan
3. Drop a number on the right pan
4. Watch the scale tip toward the heavier side
5. Try making the scale balance

**Expected Outcome**: User understands visual comparison

#### 4. Copy and Clean
**Goal**: Learn to use tools
**Concepts**: Wand (copy tool), Vacuum (delete tool)
**Steps**:
1. Create some numbers and boxes
2. Use the wand to copy an item
3. Use the vacuum to remove items
4. Practice organizing your workspace

**Expected Outcome**: User can manage workspace with tools

#### 5. Meet the Robot
**Goal**: Introduction to automation
**Concepts**: Robot widget, training mode basics
**Steps**:
1. Drag a robot from the palette
2. Click to enter training mode
3. Perform a simple action (copy a number)
4. Exit training mode
5. Let the robot replay your action

**Expected Outcome**: User understands robot training concept

### Messaging Level (tt2)

#### 6. Birds and Nests
**Goal**: Learn asynchronous messaging
**Concepts**: Bird/Nest pairs, message sending
**Steps**:
1. Switch to tt2 mode
2. Drag a bird and its nest from the palette
3. Notice how birds and nests are paired
4. Give the bird a message (drop item on bird)
5. Watch the message appear in the nest

**Expected Outcome**: User understands message passing

#### 7. Remote Communication
**Goal**: Use birds for inter-box messaging
**Concepts**: Placing nests in boxes, remote delivery
**Steps**:
1. Create two boxes
2. Put a nest in one box
3. Keep the paired bird separate
4. Send a message via the bird
5. Observe message arrival in the box

**Expected Outcome**: User can set up communication channels

## Implementation Notes

Each tutorial will be a pre-built workspace file containing:
- Initial widget layout
- Step-by-step instructions in the Workspace Notes
- Minimal widgets to avoid distraction
- Clear success criteria

Tutorials should be completable in 2-5 minutes each.
