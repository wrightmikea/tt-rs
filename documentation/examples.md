# Examples

Pre-built workspaces demonstrating tt-rs features and patterns.

## Overview

Examples are ready-to-explore workspaces that showcase what's possible with tt-rs. Unlike tutorials, examples don't have step-by-step instructions - they're meant to be examined and experimented with.

## Proposed Examples

### Arithmetic Examples

#### Calculator Layout
**Description**: A workspace organized like a simple calculator
**Widgets**: Number stacks for digits 0-9, operator numbers (+1, -1, *2, /2)
**User Level**: tt1
**Learning**: How to set up an organized workspace for calculations

#### Fraction Fun
**Description**: Demonstrates rational number arithmetic
**Widgets**: Numbers like 1/2, 1/3, 2/3, showing how fractions combine
**User Level**: tt1
**Learning**: tt-rs uses exact rational arithmetic, not floating point

### Container Examples

#### Nested Boxes
**Description**: Boxes within boxes showing hierarchical organization
**Widgets**: A 3-hole box containing smaller boxes, each with numbers
**User Level**: tt1
**Learning**: Boxes can contain other boxes

#### Data Record
**Description**: Using a box as a structured data record
**Widgets**: A labeled box representing a "person" with name, age, score
**User Level**: tt1
**Learning**: Boxes can represent structured data

### Comparison Examples

#### Sorting Setup
**Description**: Numbers arranged with scales for manual sorting
**Widgets**: Several numbers, a scale, empty boxes for "sorted" results
**User Level**: tt1
**Learning**: Using scales to compare and order values

#### Balance Puzzle
**Description**: Find the weight that balances the scale
**Widgets**: A scale with a known weight, several candidate weights
**User Level**: tt1
**Learning**: Interactive exploration of equality

### Robot Examples

#### Copy Machine
**Description**: A trained robot that copies numbers
**Widgets**: A robot trained to copy, input/output areas
**User Level**: tt1
**Learning**: Robots can automate repetitive tasks

#### Number Doubler
**Description**: A robot that doubles any number
**Widgets**: Robot trained with *2 operation, test numbers
**User Level**: tt1
**Learning**: Robots can perform calculations

### Messaging Examples (tt2)

#### Mailbox System
**Description**: Bird/nest pair demonstrating message queuing
**Widgets**: A bird, its nest, several messages to send
**User Level**: tt2
**Learning**: Messages queue in the nest

#### Two-Way Communication
**Description**: Two bird/nest pairs for bidirectional messaging
**Widgets**: Two birds, two nests arranged for back-and-forth
**User Level**: tt2
**Learning**: Setting up communication channels

## Implementation Notes

Each example will be a workspace file containing:
- Pre-arranged widgets demonstrating the concept
- Brief description in Workspace Notes
- No step-by-step instructions (explore freely)
- Representative of a common pattern or use case

Examples should be immediately interesting to explore.
